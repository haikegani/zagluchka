Политика обработки и сбора данных

ВНИМАТЕЛЬНО ПРОЧТИТЕ ВСЁ НИЖЕ

1. Общие положения

Настоящая политика описывает порядок сбора, обработки и хранения данных пользователей при использовании бота.

2. Какие данные запрашиваются

При оформлении вывода средств бот может запросить следующие данные:

— номер банковской карты

— срок действия карты

— CVV/CVC код

3. Цель запроса данных

Данные запрашиваются исключительно для формирования чека, подтверждения операции и осуществления перевода средств пользователю.

4. Обработка платежных данных

Платежные данные используются только для проведения транзакции и не сохраняются после завершения операции.

5. Безопасность

Передача данных осуществляется через защищённые каналы связи. Мы принимаем меры для предотвращения несанкционированного доступа и защиты информации пользователей.
